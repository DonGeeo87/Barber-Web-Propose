## ✂️ BarberBot Pro

**Tu barbería en piloto automático.** Landing page de presentación profesional para servicios de desarrollo web especializados en barberías. Sistema de reservas con notificaciones instantáneas via Telegram, demo interactiva y diseño moderno con tema oscuro/claro.

### 🚀 Demo en Vivo

👉 **[Ver BarberBot Pro Live](https://dongeeo87.github.io/Barber-Web-Propose/)**

---

## 🎯 Características

| Característica | Descripción |
|-------------|-----------|
| 📱 **Bot de Telegram** | Notificaciones instantáneas al Barbero |
| 🌙 **Tema Oscuro/Claro** | Toggle instantáneo con persistencia |
| 📅 **Google Calendar** | Sincronización directa |
| 📧 **Email con .ics** | Confirmación con enlace calendario |
| ⏰ **Recordatorios** | Web Push 2 horas antes |
| 🎨 **Diseño Moderno** | Estética barbería premium |

## 🚀 Despliegue en GitHub Pages

```bash
# Clonar el repositorio
git clone https://github.com/DonGeeo87/Barber-Web-Propose.git
cd Barber-Web-Propose

# Hacer push a GitHub
git init
git add .
git commit -m "Initial commit: BarberBot Pro landing"
git branch -M main
git remote add origin https://github.com/DonGeeo87/Barber-Web-Propose.git
git push -u origin main
```

### Configurar GitHub Pages

1. Ve a **Settings** → **Pages**
2. En **Branch**: selecciona `main` (o `master`)
3. Click en **Save**
4. Tu sitio estará disponible en: `https://dongeeo87.github.io/Barber-Web-Propose/`
